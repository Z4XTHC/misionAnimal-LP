## 🚀 Descripción del Proyecto
Este sitio web es una **Single Page Application (SPA)** diseñada para optimizar la experiencia de los usuarios que buscan interactuar con la ONG. El diseño se basa en un enfoque **Minimalista Material UI**, priorizando la legibilidad y la facilidad de uso en dispositivos móviles.

### 🎨 Identidad Visual
- **Color Primario:** `#7DBFB8` (Verde Menta/Cian).
- **Color Secundario:** `#8D735B` (Marrón Tierra).
- **Estilo:** Flat Design (Sin degradados ni efectos 3D).

---

## 🏗️ Estructura del Sitio
La navegación está organizada en secciones clave para cubrir todas las necesidades de la organización:

| # | Sección | Propósito |
| :--- | :--- | :--- |
| 1 | **⚓ Navbar** | Navegación rápida y persistente. |
| 2 | **⚡ Hero** | Mensaje principal y slogan institucional. |
| 3 | **👥 Quiénes Somos** | Nuestra historia y valores. |
| 4 | **🤝 Cómo Ayudamos** | Pilares: Adopción, Castración y Tránsito. |
| 5 | **🙋 Voluntariado** | Guía interactiva para sumarse a la causa. |
| 6 | **📅 Castraciones** | Gestión directa de turnos por WhatsApp. |
| 7 | **🎓 Educación** | Programas de activismo y charlas comunitarias. |
| 8 | **📰 Noticias** | Calendario de efemérides animalistas. |
| 9 | **🚨 Denuncias** | Protocolos legales contra el maltrato animal. |
| 10 | **📞 Contacto** | Redes sociales y ubicación geográfica. |

## ♿ Estándar de Accesibilidad (A11y) - Mango Software
Siguiendo los lineamientos de **Mangosoft**, este sitio garantiza la inclusión mediante:
*   **Modo Oscuro:** Adaptación visual automática.
*   **Escalado Dinámico:** Herramienta de aumento de fuente sin pérdida de layout.
*   **Semántica Estricta:** Uso correcto de etiquetas `<main>`, `<section>`, y `<header>`.
*   **Navegación Asistida:** Atributos ARIA y foco visible para teclados.
